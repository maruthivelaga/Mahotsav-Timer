# Vignan Mahotsav - Coming Soon Timer

A React countdown timer application for the Vignan Mahotsav event.

## Getting Started

1. Install dependencies:
```bash
npm install
```

2. Run the development server:
```bash
npm run dev
```

3. Build for production:
```bash
npm run build
```

4. Preview production build:
```bash
npm run preview
```

## Features

- Animated preloader
- Countdown timer to event date (Feb 5, 2026)
- Beautiful background with decorative elements
- Responsive design
- Smooth animations

## Tech Stack

- React 18
- Vite
- CSS3 Animations
